My Projects

Real-Time 3D Product Configurator (Lamps)

A Unity WebGL (C#) and JavaScript-based 3D lamp configurator. Users assemble custom lamps in real time, with automated EmailJS delivery of the final configuration, including user data and a generated SVG technical drawing.

ReplayVR Video Platform

A responsive web application built with Vue.js and TypeScript for ReplayVR. The platform enables users to upload and stream video files through an API-driven workflow, where content is processed and dynamically displayed in a dropdown interface. Users can create and manage personalized playlists, supported by a fully implemented authentication system.

Interactive 3D Color Configurator

A web-based visualization tool built with HTML, CSS, and JavaScript to showcase color textures. Using the Sketchfab API, users can dynamically apply and modify colors on a 3D model in real time. Includes a search system to quickly filter and find specific textures.

VR Training Platform for Groundwater Purification

A role-based training platform built with Vue.js and TypeScript for groundwater purification. Teachers can assign tasks to students, while the system communicates in real time with VR headsets via WebSockets to track performance and generate reports.

Interactive Energy Simulation (Multiplayer)

A multiplayer Unity (C#) application developed for TotalEnergies to demonstrate energy conversion through interactive systems. Features an offline-first architecture that switches between multiplayer and single-player modes based on internet availability.

Multiplayer VR Lighting Configurator (LednLux)

A multiplayer Unity (C#) configurator for LednLux, featuring a lobby system that allows users to join shared sessions and collaboratively visualize and customize LED lighting solutions in an immersive 3D environment.

Interactive Agriculture Simulation (Kubota)

A multiplayer Unity (C#) application developed for Kubota to showcase oil products within an interactive simulation environment. Users can explore and interact with systems in real time.

Multiplayer Logistics Training Simulator

A multiplayer Unity (C#) training application focused on logistics scenarios, featuring interactive tasks and real-time collaboration between users in a simulated environment.
