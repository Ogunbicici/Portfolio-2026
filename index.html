My Projects

3D Lamp Configurator

Unity WebGl (C#) and JavaScript based 3D lamp configurator. Users assemble custom lamps in real time, with automated EmailJS delivery of the final configuration, including user data and a generated SVG technical drawing.

ReplayVr Website

A responsive web application built with Vue.js and TypeScript for ReplayVR. The platform enables users to upload and stream video files through an API driven workflow, where files are processed and dynamically displayed in a dropdown interface on the home screen. Users can also create and manage personalized video playlists. The project includes a fully implemented authentication system for secure user access..

Web Color Configurator

A web based visualization tool built with HTML, CSS, and JavaScript to showcase color textures. Using the Sketchfab API, users can dynamically apply and modify colors on a 3D model in real time. The project also includes a search system to quickly filter and find specific textures, improving overall user experience.

Training groundwater purification

Vue.js and TypeScript role based training platform for groundwater purification, featuring teacher/student assignments and real time VR communication via WebSockets with performance reporting.

Interactive Energy Simulation (Multiplayer)

A multiplayer application built in Unity (C#) that demonstrates how energy is converted and utilized. Users can interact with buttons, grabbable objects, and various in-game systems to explore energy processes in an interactive way. The application was developed for TotalEnergies.
